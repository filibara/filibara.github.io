Operatory w JavaScript

Arytmetyczne

Dodawanie: 2 + 3 = 5

Odejmowanie: 5 - 2 = 3

Mnożenie: 4 * 2 = 8

Dzielenie: 8 / 2 = 4

Modulo: 7 % 3 = 1

Logiczne

true && false = false

true || false = true

!true = false

Bitowe

5 & 1 = 1

5 | 1 = 5

5 ^ 1 = 4

~5 = -6

Przypisania

x = 5

x += 2 (dodanie i przypisanie)

x -= 1 (odjęcie i przypisanie)

Porównania

5 == "5" → true

5 === "5" → false

7 > 3 → true

7 <= 3 → false

Łańcuchowy

Łączenie stringów:

"Hello" + " World" = "Hello World"
