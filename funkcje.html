parseInt(string, [radix])

Parsuje string na liczbę całkowitą. Radix opcjonalny (podstawa, np. 10).
parseInt("123") → 123
parseInt("08") → 8 (use radix 10 to be safe)
parseInt("10px") → 10

parseFloat(string)

Parsuje liczbę zmiennoprzecinkową z początku stringa. Zwraca NaN jeśli nie zaczyna się liczbą.
parseFloat("45.67") → 45.67
parseFloat("3.14abc") → 3.14
parseFloat("abc3.14") → NaN

Number(value)

Konwertuje wartość na liczbę. Wymaga, żeby cały string był liczbą (trimowane). Zwraca NaN dla nieprawidłowych wartości.
Number(" 123 ") → 123
Number("12e2") → 1200
Number("10px") → NaN

isNaN(value)

Sprawdza, czy wartość jest NaN. Uwaga: przed ES6 trzeba być ostrożnym; lepiej używać Number.isNaN.
isNaN("abc") → true
isNaN("123") → false
Number.isNaN(NaN) → true

safeParse(value) — custom helper

Moja funkcja: próbuje Number(), potem parseFloat(), zwraca obiekt z metodą i wartością; przy NaN proponuje fallback.
safeParse(" 12.34 ") → { method: "Number", value: 12.34 }
safeParse("12px") → { method: "parseFloat", value: 12 }
safeParse("abc") → { method: "none", value: NaN }
