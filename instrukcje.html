if • switch • for • while • do...while • ternary

Przykłady

Każdy przykład można uruchomić i podejrzeć snippet kodu.

if / else

Instrukcja warunkowa w trzech wariantach: zwykły, z else i bez klamerek.
if (warunek) { ... } else { ... }

switch

Wybiera blok kodu na podstawie wartości zmiennej.
switch(day) { case 1: ... }

for

Pętla z licznikiem.
for (let i=0; i<5; i++) { ... }

while

Wykonuje blok dopóki warunek jest spełniony.

do...while

Wykonuje blok co najmniej raz, a potem sprawdza warunek.

?: (operator trójargumentowy)

Zapis warunku w skróconej formie.
warunek ? wynik1 : wynik2
