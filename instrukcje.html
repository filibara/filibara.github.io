Instrukcje sterujące w JavaScript

Instrukcja if

Trzy warianty użycia instrukcji if:

  • Zwykły if
  • if...else
  • if bez klamerek
if (warunek) { // kod gdy warunek jest prawdziwy } else { // kod gdy warunek jest fałszywy } if (warunek) // kod bez klamerek

Instrukcja switch

Pozwala porównywać jedną wartość do wielu przypadków.

switch (zmienna) { case 1: // kod break; case 2: // kod break; default: // kod domyślny }

Pętla for

Pętla używana do wykonywania kodu określoną liczbę razy.

for (let i = 0; i < liczba; i++) { // kod wykonywany w pętli }

Pętla while

Wykonuje kod dopóki warunek jest prawdziwy.

while (warunek) { // kod wykonywany dopóki warunek true }

Pętla do...while

Pętla zawsze wykona się przynajmniej raz, zanim sprawdzi warunek.

do { // kod wykonywany raz na start } while (warunek);

Instrukcja przetwarzania warunkowego (? :)

Skrócona forma if...else.

warunek ? wartość_jeśli_true : wartość_jeśli_false;
